# diffbot

基于ROS2的差速移动机器人。主要功能：

- 与安卓上位机交互：上位机显示交互界面，地图等。
- Nav2的调用：导航，导航禁区，航点跟踪等。
- 传感器：烟雾报警，明火报警，空气质量检测，气温，血氧检测等。
- 云台摄像头：唤醒跟踪人脸，主动检测问候等
- 声场定位：唤醒面对说话者。
- 语言交互：自然语音聊天，自然语言动作控制。
